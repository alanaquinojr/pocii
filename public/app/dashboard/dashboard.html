Apliacaçao demo usando AngularJS com Node Js, Neo4J e Express. Use 'guest'/'guestpassword' para acessar o sistema.
Adicione ou remova pontos de ligação (blocos, pares, dispositivos, etc..). Quando um novo elemento for adicionado ele será automaticamente adicionado ao menu.
Depois de criar os pontos de conexão, informe o circuito que os interliga. Escolha dois pontos e crie a ligaçao.
